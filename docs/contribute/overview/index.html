Overview

How to Join#

Request an invitation by sending an email stating your name, occupation and provide a brief explanation of what excites you about being part of the Njinius community. Also, for verification purposes, please include a link to your Twitter, Linked-in or Medium profile. You can rest assured that this information will not be shared with anyone and will not be retained post verification.

Once verification has been completed, NJinius will send you an invitation link to our slack workspace. When you get the invite, jump on the introduction channel and tell us a little about yourself. You will see a few introductions already in there if you're unsure of what to say - don't be shy ðŸ˜„

Be sure to read our code of conduct and explore our slack channels. There's a quick guide to using slack here

Contributing#

Join us and contribute to the community. The various ways to contribute are described below.

Slack Community#

You don't have to get left out of the conversation. You can join in any conversation, from business topics to software, books, and even general chat. NJinius slack provides for all of these, simply join the relevant channel, and you're good to go.

Recommended channels#

If you are unsure where to start, we recommend the following channels

  • #njinius_general_help
  • #njinius_challenges
  • #technology_general
  • #announcements
  • #people_leadership

Content Contributions#

Njinius covers several exciting topics, be it in software, business, or books. We understand that people have a diverse range of knowledge, expertise and interests. If you would like to introduce a new topic to NJinius, you can follow the steps in the contribution Direct Contribution Guide. The Guide includes instructions on how to submit changes directly to the NJinius website.

If you do not wish to go through the Guide's process, please send your request via email. Your request will be reviewed, and if approved, the requested changes will be made to the website.

Website Contributions#

There are different ways to contribute to Njinius, one of which is by contributing directly to the website. This includes anything from the UI(user interface) to the site's layout, the colour scheme and appearance.

This can be done by creating a GitHub account and cloning the repository. Once you have done that, you can make the necessary changes then submit them for review. To make these changes, there are a few pre-requisites;

  • Github Account
  • Git
  • Node.js
  • IDE to edit the code, we recommend Microsoft Visual Studio Code

Once you have cloned the repository, you can make the changes you would like and submit them for review. Instructions on getting the above requisites, cloning the repository and submitting for review can be found in the Direct Contribution Guide section.

Host Live Sessions#

Njinius hosts a variety of live discussions periodically. Everyone is welcome to present or participate in a session.

The easiest way to participate in live sessions or host one is to join our NJinius slack workspace, specifically the #live-sessions channel. Alternatively, to host a session, send us an

email, with the subject "Host Live Session", along with a synopsis of the talk.

NJinius will get back to you within 24 hours, with follow up steps and to confirm the available time slots.
