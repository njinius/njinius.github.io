Overview#

C++ is a popular general-purpose programming language designed to provide high performance, efficiency, and flexibility. All major operating systems are written in a combination of C and C++. Developers with C++ skills are frequently in demand and hold some of the industry's most high paying jobs. The language is commonly used in applications that require high performance, such as image processing programs and the hardware level in embedded systems.
